# Bookshop Service

Following commands can be used:

Run Tests
### `./gradlew clean test`

Run Code Coverage Validation
### `./gradlew jacocoTestCoverageVerification`

Build a Jar
### `./gradlew clean build`

Pull latest for all docker images
### `docker-compose pull`

Run all containers
### `docker-compose up -d`

Stop all containers
### `docker-compose down`
